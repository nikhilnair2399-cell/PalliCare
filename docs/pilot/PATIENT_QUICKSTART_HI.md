# पैलीकेयर — त्वरित प्रारंभ गाइड
## PalliCare — Quick Start Guide

---

### ऐप इंस्टॉल करें / Install the App

1. QR कोड स्कैन करें → ऐप डाउनलोड होगा
2. "Install" बटन दबाएँ
3. ऐप खोलें → भाषा चुनें (हिन्दी / English)

---

### दर्द दर्ज करें / Log Your Pain

| चरण | क्या करें |
|------|----------|
| 1 | होम स्क्रीन पर **"Log"** बटन दबाएँ |
| 2 | दर्द का स्तर चुनें: **0** (कोई दर्द नहीं) से **10** (सबसे तेज दर्द) |
| 3 | शरीर के नक्शे पर दर्द की जगह दबाएँ |
| 4 | **"Save"** दबाएँ — बस हो गया! |

**त्वरित लॉग**: सिर्फ दर्द का अंक दर्ज करने के लिए "Quick Log" चुनें (3 टैप में पूरा)

---

### दवाई ट्रैक करें / Track Medications

| चरण | क्या करें |
|------|----------|
| 1 | होम स्क्रीन पर दवाई की पट्टी देखें |
| 2 | दवाई लेने के बाद **हरा बटन** ✓ दबाएँ |
| 3 | दवाई छूट गई तो **लाल बटन** ✗ दबाएँ |

**रिमाइंडर**: ऐप दवाई के समय पर सूचना भेजेगा

---

### रोज़ाना करें / Daily Routine

| समय | क्या करें |
|------|----------|
| **सुबह** 🌅 | मॉर्निंग चेक-इन: दर्द + मूड + नींद दर्ज करें |
| **दवाई के समय** 💊 | दवाई लेने पर ✓ दबाएँ |
| **शाम** 🌙 | दर्द का अंक दर्ज करें |
| **जब भी दर्द बढ़े** ⚡ | ब्रेकथ्रू पेन लॉग करें |

---

### मदद चाहिए? / Need Help?

| स्थिति | क्या करें |
|--------|----------|
| **आपातकालीन** 🚨 | **112** पर कॉल करें |
| **पैलिएटिव हेल्पलाइन** ☎️ | 011-2658-8500 |
| **ऐप में समस्या** 📱 | Settings → About → Report Issue |
| **ऐप सीखना है** 📖 | Learn सेक्शन में गाइड पढ़ें |

---

### याद रखें / Remember

- ✅ **हफ्ते में कम से कम 3 बार** दर्द दर्ज करें
- ✅ **दवाई समय पर** लें और ऐप में दर्ज करें
- ✅ **ऑफलाइन भी काम करता है** — इंटरनेट आने पर डेटा सिंक होगा
- ❌ **यह ऐप डॉक्टर की जगह नहीं है** — आपातकालीन में डॉक्टर से संपर्क करें
